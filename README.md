# samaadhaan

1. Samaadhaan is a Government Complaints Application Designed and Developed using Flutter Framework,
Firebase for the backend including CRUD Operations, multiple Flutter Packages for diﬀerent features.
2. Samaadhaan is built for a hassle free complaint registration as it supports all types of complaints on single
platform, a user can select the particular Ministry under which his/her complaint falls and can register
complaints.
3. Key Features:
Mobile OTP Verification, Human Verification, Multilingual Speech-to-text Support, Image Upload
